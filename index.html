Questions

FAQ

Which Python and Django versions are supported?
Python 3.10, 3.11, 3.12, and 3.13. Django 4.2 LTS. Tested on all four Python versions. Django 5.x migration guide coming in a future update.
How do I set up Google or Facebook login?
Configure credentials in Google Cloud Console or Facebook Developers. Add the client ID and secret to your .env file. Leave them empty to hide the buttons completely — no code changes needed.
Can I change the theme / branding?
Yes. Edit the CSS variables in iam.css to change colors. Set SITE_NAME and SITE_LOGO in settings.py. The dark theme is built with variables so a full rebrand takes less than 5 minutes.
Does it work with PostgreSQL?
Yes. SQLite for development (zero config), PostgreSQL for production. Just change the DB_ENGINE and DB_* variables in your .env file.
Is Arabic (RTL) really fully supported?
Yes. 282 Arabic strings pre-translated. Separate rtl.css for right-to-left layout overrides. Language switch is instant via the API — no page reload needed.
